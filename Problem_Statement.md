Idea

Ek app jahan user internet se koi bhi cheez save kare aur system automatically usko organize, relate aur resurface kare.

Example:

User save karta hai
• article
• tweet
• image
• youtube video
• pdf

App automatically karta hai:

• tag suggestions
• topic clustering
• knowledge graph
• related items suggest
• resurfacing (“2 months ago you saved this”)

Key Features

• browser extension save tool
• semantic search
• graph visualization
• AI tagging (optional)
• memory resurfacing
• collections
• highlight system

Tech Complexity

Frontend
• React / Next
• graph visualization (d3.js)

Backend
• Node / Python microservice
• embedding search
• queue workers

Infra
• object storage
• vector DB
